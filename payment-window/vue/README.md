# vue-project

## 기본설명
```
VUE는 결제화면을 호출하는 부분만 참고해야 합니다.
VUE는 Front side 언어이기 때문에 승인 API의 Authorization이 외부에 노출될 수 밖에 없습니다.
승인 API를 호출 로직은 Server side 언어를 사용하여 구현해야 합니다.

```
